# MIR Engine

A high-performance, ultra-lightweight C++20 and SDL3 2D game engine built around the Entity-Component-System (ECS) architecture and zero-allocation containers.

### Quick Start / 빠른 시작
```bash
# Configure and build the project
xmake f -m release
xmake

# Run the game engine
xmake run mirengine

# Run unit tests
xmake build mirengine-tests
xmake run mirengine-tests
```
